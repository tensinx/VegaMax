이미지 기반 ADB 매크로 툴 — 손쉬운 자동화, 강력한 설정

다운로드

⤓ VegaMax 다운로드
버전: v1.0.0

설치 가이드

  1. 다운로드한 exe 파일을 실행하세요. (관리자 권한 권장)
  2. 설치 후 프로그램을 실행하고 설정 ▶︎ ADB 경로에서 adb.exe 경로를 지정하세요.
  3. 이미지 폴더를 선택하고 타겟 이미지를 등록한 뒤 매크로를 실행하세요.

시스템 요구사항

  • Windows 10/11 (64-bit 권장)
  • Python 라이브러리(배포판 포함) 또는 독립 실행형 exe
  • LDPlayer 같은 에뮬레이터 및 ADB 환경

FAQ

Q: ADB가 인식 안돼요
A: 설정 > ADB 경로에서 adb.exe를 정확히 지정하세요. 에뮬레이터가 켜져 있어야 합니다.
Q: 이미지 매칭 정확도를 높이고 싶어요
A: 타겟 이미지의 임계값(threshold)을 높이고 해상도/크롭된 이미지를 사용하세요.

핵심 기능

이미지 탐지 & 자동 클릭
ADB 직접 제어 (탭/롱프레스/드래그)
쿨다운 및 임계값 설정
프로파일 저장/불러오기
